PyQt5 database application generator using django like models

Requirements : python 3.6, PyQt5

```bash
python3 -m venv ~/venv/sofosvenv
source ~/venv/sofosvenv/bin/activate
pip install sofos

sofos-project sofos_test
cd sofos_test
python main.py
```
